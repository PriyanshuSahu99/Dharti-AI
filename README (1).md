# 🌱 Dharti AI

**Dharti AI** is an AI-powered smart agriculture platform that helps
farmers make informed decisions using Machine Learning, Weather
Intelligence, Disease Detection, Market Analysis, and an AI Chatbot.

The project integrates **FastAPI** as the backend and **Streamlit** as
the frontend.

## 🚀 Features

-   🌾 Smart Crop Recommendation using ML
-   🌦 Live Weather Intelligence (OpenWeather API)
-   🌿 Plant Disease Detection
-   📈 Market Intelligence
-   🤖 AI Farming Assistant
-   📄 Report Generation

## 🛠 Tech Stack

### Backend

-   FastAPI
-   Python
-   Scikit-learn
-   Pandas
-   NumPy
-   Uvicorn

### Frontend

-   Streamlit
-   HTML/CSS

### APIs

-   OpenWeather API

## ⚙️ Installation

``` bash
git clone https://github.com/PriyanshuSahu99/Dharti-AI.git
cd Dharti-AI
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## ▶️ Run the Project

### Backend

``` bash
cd backend
uvicorn main:app --reload
```

### Frontend

``` bash
cd frontend
streamlit run app.py
```
